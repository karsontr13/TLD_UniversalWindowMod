# TLD_UniversalWindowMod
A mod that allows in-game windows to simulate the outside weather. Now you can watch snow falling outside.

The mod is currently in early development. However, I'm releasing it to lay a solid foundation for its development. Please report any bugs you notice.
Currently, only window images of the hunter's cabin and camp office in the Mystery Lake area are available. Since the mod is still in the early stages of development, it will take some time before all areas are released. New interiors and areas are being added continuously.
